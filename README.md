# 📰 The Blox Bulletin

Site d'actualités pour Blox Fruits — les bulletins hebdomadaires traduits en français.

## Structure des fichiers

```
blox-bulletin/
├── index.html              ← Page d'accueil (liste des bulletins)
├── fonts/
│   └── LuckiestGuy-Regular.ttf   ← À copier depuis ton ordinateur
├── shared/
│   └── img/
│       └── logo.png        ← Logo Blox Fruits (Logo_de_Blox_Fruits.png)
├── 001/
│   ├── bulletin-001.html
│   └── img/                ← Images du bulletin 001
├── 002/
│   ├── bulletin-002.html
│   └── img/                ← Images du bulletin 002
├── 003/
│   ├── bulletin-003.html
│   └── img/                ← Images du bulletin 003
└── 004/
    ├── bulletin-004.html
    └── img/                ← Images du bulletin 004
```
